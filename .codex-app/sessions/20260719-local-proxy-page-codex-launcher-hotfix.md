# Session: Local Proxy Page And Codex Launcher Hotfix

Date: 2026-07-19 UTC

## Goal
- User reported the local relay management page could not open even after `重启代理` reported success.
- User requested a proxy startup status display in the relay dialog.
- User reported Codex failed after install with `/root/.openclaw/bin/codex: 23: /root/.config: Permission denied` and ChatGPT account bootstrap auth error `plan type is required`.

## Repo Facts Read
- Repo root: `/storage/emulated/0/ZeroTermux/开发/openclaw-termux-zh-5.5`.
- Current APK candidate before this fix: `7.4 / 193`, GitHub Actions run `29661374383`.
- api2py `index.html` references `/static/lib/tailwind.js` and `/static/lib/lucide.js`.
- The user's source `/storage/emulated/0/ZeroTermux/开发/api2py/public/static/lib/` contains both JS files, but the project asset copy did not.
- Codex launched from `/root/.openclaw/bin/codex`, generated by `CliApiConfigService`, not only from `/usr/local/bin/codex` generated by the install script.

## Changes Made
- Bundled `flutter_app/assets/api2py/public/static/lib/tailwind.js` and `lucide.js` from the user's api2py source tree.
- Added `assets/api2py/public/static/lib/` to `flutter_app/pubspec.yaml`.
- `LocalApiProxyService` now exposes `LocalApiProxyStatus`, checks health, root management page, and both static JS resources, and verifies these files after asset sync.
- Proxy startup commands now require `/`, `/static/lib/tailwind.js`, and `/static/lib/lucide.js` to respond before reporting success.
- `LocalApiProxyBrowserScreen` checks/starts the service before loading the WebView and displays service errors in-app.
- `中转代理` dialog now shows running/manageable status, refreshes status, and updates after restart.
- Managed CLI launchers now generate a single-line `mkdir -p` command so `/root/.config` cannot become a standalone command after line-continuation damage.
- Codex configs routed through the local 9999 proxy now get placeholder `OPENAI_API_KEY=openclaw-local-proxy` when no direct per-tool key is stored, forcing API-key auth instead of ChatGPT account bootstrap.
- `THIRD_PARTY_NOTICES.md` records Tailwind/Lucide api2py bundled paths; ESLint ignores the vendored api2py static JS directory.
